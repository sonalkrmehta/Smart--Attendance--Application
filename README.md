------> VisionTrack AI: Smart Attendance System <---------

VisionTrack AI is a robust, full-stack biometric attendance solution that utilizes state-of-the-art Deep Learning to automate attendance tracking. By capturing live video feeds, the system detects faces, generates 128D embeddings, and matches them against a pre-trained database in real-time.

✨ Key Features

🚀 Real-Time Face Recognition: Low-latency detection and identification using Dlib's ResNet-128 model.

👤 Instant Enrollment: Seamlessly register new users by capturing face samples directly from the web dashboard.

📊 Live Analytics Dashboard: Interactive UI showing real-time detection logs, accuracy metrics, and total daily attendance.

🛠️ DevOps Integrated: Fully containerized with Docker and orchestrated with Docker Compose.
